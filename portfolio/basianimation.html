Basic Animation

This theme aims at introducing concepts of basic animation, classic storytelling and character design as well as work with CSS animations and enhanced interactivity through Javascript.

Gained skills of planning and conducting user tests of digital media productions, including selection of the right user test for a given digital media production

Simple animation


In this project we have been introduced to Javascript, CSS animation or transition.

The main goal of this project was creating a website containing a character and buttons.

When the user clicks a button, an animation starts.i created different animations such as jump, rotate or walk by using CSS keyframes animation.

From Javascript we used function,querySelector and addEventlistener.It was a bit difficult for me in the beginning but with teachers and classmates help i was able to finish this project.


The relationship goal story


This project included creating a story with clear message and describing the genre/mood of the story. The story had to have a dramaturgic curve which consists of a prelude, point of no return, climax and end - fade out.

The storyboard had to include all the dramaturgical elements (e.g sounds,time, movement) and we had to create a narrative curve too. I created a story called " The relationship goals".

It was about couple Linda and Tom who are getting ready to go out, but Linda keeps complaing that she has nothing to wear.When she finally found an outfit, Tom meanwhile turned into skeleton.



Interactive animation


This interactive story has to consist of at least 3 chained events that bring the story forward and we had to use at least 3 different event types (click, ended, animationend) and the scene should included the sound too. For animating our characters we used CSS keyframes animation. From Javascript we used document.getElementByID,console.log or addEventListener.

We made our project in group of three and it was quite difficult for us. We needed some classmates help but after we were able to make it and finish it. There is a lot to improve on this project but i was able to learn how to make an interactive animation using javascript.The story included sequence diagram and storyboard as well.

Story about Pinocchio


This was a group project,we were a group of 7. We received the style,music and we have to create the characters in illustrator and animate it using CSS and Javascript. Our style was surrealism and we decided to make a story about Pinocchio.

My role was designer and UX. I designed the last "trippy" scene and done the user testing, which included peer review.

We received very good feedback from teachers about this project and were very glad about it.
